lastIndexOf

lastIndexOf(arr, value, [fromIndex]) ⇒ number

Return the last index of the matched value, -1 if nothing matches

  • comparison are done after handling with JSON.stringify

Kind: global function

Param Type Description
arr Array.<any> the array
value any the array
[fromIndex] number optional, if specified, the search will start at the specified index and from back to front, negative value is also supported

Example 

const arr = [1, '2', 'c', '2']
console.log(lastIndexOf(arr, '2')) // 3
console.log(lastIndexOf(arr, '2', 3)) // 3
console.log(lastIndexOf(arr, '2', 4)) // -1
